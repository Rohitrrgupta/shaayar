ruTha tha tujh se yaani khud apni khushi se main
phir us ke baad jaan na ruTha kisi se main

banhon se meri vo abhi rukhsat nahin hua
par gum huun intizar men us ke abhi se main

dam-bhar tiri havas se nahin hai mujhe qarar
halkan ho gaya huun tiri dilkashi se main

is taur se hua tha juda apni jaan se
jaise bhula sakunga use aaj hi se main

rukhsat hua huun tere liye dil-gali se main
tu hi harim-e-jalva hai hangam-e-rang hai

janan bahut udaas huun apni kami se main
kuchh to hisab chahiye aine se tujhe

lunga tira hisab miri jaan tujhi se main
